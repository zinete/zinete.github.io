git 代码提交验证问题

配置 gpg 之后如果提交代码失败，需要在终端执行 export GPG_TTY=$(tty)
然后再次提交，按照提示输入 gpg 配置的密码即可

gpg --full-generate-key 交互式命令创建新的 gpg key

gpg --list-keys 查看创建的所有 gpg key

gpg --armor --export zhenghui@zinete.com 查看指定邮箱 gpg 公钥
